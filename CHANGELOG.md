# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2017-12-30
### Added
- Adapt script for a class file (you don't need to edit the script manually to use it anymore).
- Create-Delete Demo Example.
- Improved Error Trapping.
- Documentation.

### Changed
- Migrate registry manipulation logic to RegClassVBS.

## [1.0.5] - 2013-11-15
### Added
- Now able to update NTUSER.DAT and/or USRCLASS.DAT (HKCU and/or HKCR).

## [1.0.4] - 2013-08-25
### Added
- Added ability to delete keys. 

## [1.0.3] - 2013-04-23
### Added
- Added ability to write REG_BINARY values.

## [1.0.2] - 2013-04-11
### Fixed
- Fixed bug where it wouldn't work when run by SYSTEM account.

## [1.0.1] - 2013-03-28
### Changed
- Huge code cleanup and bug fixes.

## 1.0.0 - 2012-01-13 
### Added
- Initial release.
